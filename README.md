# Nakanishi's dotfiles

## Installation

    git clone https://github.com/NakanishiTetsuhiro/dotfiles
    cd ~/dotfiles
    ./install.sh -d

or

    Warning: It does not work now
    curl -L raw.github.com/NakanishiTetsuhiro/dotfiles/master/install.sh | bash -s -i
