# Welcome to Nakanishi's dotfiles

## Installation

    git clone https://github.com/NakanishiTetsuhiro/dotfiles $HOME/dotfiles
    ansible-playbook $HOME/dotfiles/ansible/deploy.yml -i $HOME/dotfiles/ansible/hosts
    omf theme default
